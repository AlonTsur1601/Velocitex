## Title

Velocitex | A 3D Momentum Puzzle Game Built with Codex and GPT-5.6

## Description

Velocitex is a Windows 3D momentum-puzzle game set inside a candy machine. Guide a candy through 28 rooms where slopes, surfaces, launches, turns, and stopping distance all matter.

This Build Week demo shows real gameplay from the Windows build, including the escalating momentum mechanics, customization, advancements, saves, and accessibility settings.

Velocitex was developed in Godot 4.7 .NET with C#. Codex supported the implementation and verification workflow across gameplay systems, physics and route investigation, build/export automation, and repeatable tests. GPT-5.6 was used through Codex to help reason through design and engineering constraints, refine implementation work, and expand verification coverage. Final product, design, and engineering decisions remained with the creator.

Play the Windows build: https://github.com/AlonTsur1601/Velocitex/releases/latest
Source code and testing instructions: https://github.com/AlonTsur1601/Velocitex

Controls: WASD or Arrow Keys to move, Mouse to look, C to switch camera, Middle Mouse to zoom, E to interact, Esc for the pause menu.
